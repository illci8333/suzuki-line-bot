## JLPT N5 必考文法句型（80 個）

### 一、基礎助詞與判斷句

| 句型 | 用法 | 例句 | 翻譯 |
|------|------|------|------|
| ～は～です | 提示主語（是...） | 私は学生です。 | 我是學生。 |
| ～も | 也 | 私も学生です。 | 我也是學生。 |
| ～の | 名詞修飾名詞（的） | これは私の本です。 | 這是我的書。 |
| ～を | 動作的對象（受詞） | パンを食べます。 | 吃麵包。 |
| ～に（時間） | 動作發生的具體時間 | 七時に起きます。 | 七點起床。 |
| ～に（場所） | 存在的場所、目的地 | 部屋に机があります。 | 房間裡有桌子。 |
| ～に（對象） | 動作的對象（給、對...） | 友達に電話をかけます。 | 給朋友打電話。 |
| ～へ | 移動的方向 | 日本へ行きます。 | 去日本。 |
| ～で（場所） | 動作進行的場所 | 図書館で勉強します。 | 在圖書館學習。 |
| ～で（手段） | 工具、交通工具、手段 | 箸で食べます。 | 用筷子吃。 |
| ～と | 和、一起 | 友達と映画を見ます。 | 和朋友看電影。 |
| ～か | 疑問助詞 | これは何ですか。 | 這是什麼？ |
| ～ね | 確認、徵求同意 | いい天気ですね。 | 天氣真好呢。 |
